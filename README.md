# Ringkasan Tugas 4, 5, dan 6

Secara keseluruhan, ketiga tugas ini memberikan pemahaman bertahap tentang penggunaan Python.

- [Tugas 4](https://github.com/Shlyn/tugas_python_ai_b10/blob/main/tugas4.py) berfokus pada struktur data bawaan seperti *list*, *tuple*, *set*, dan *dictionary* beserta berbagai operasinya.
- [Tugas 5](https://github.com/Shlyn/tugas_python_ai_b10/blob/main/tugas5.py) memperkenalkan pembuatan fungsi dan konsep *Object-Oriented Programming (OOP)* sederhana melalui kelas untuk mengelola data siswa.
- [Tugas 6](https://github.com/Shlyn/tugas_python_ai_b10/blob/main/tugas6.py) menggabungkan penggunaan library **NumPy** dan **pandas** untuk analisis data dan menerapkan OOP dalam pengelolaan nilai mahasiswa. Ringkasan analisa dan statistiknya dapat dilihat pada [ringakasan_tugas6.txt](https://github.com/Shlyn/tugas_python_ai_b10/blob/main/ringkasan_tugas6.txt).

Melalui ketiga tugas ini, kita mendapatkan pemahaman tentang:
- Pengelolaan data secara efisien menggunakan struktur data Python
- Penulisan kode yang lebih terstruktur dengan fungsi dan OOP
- Pemanfaatan library Python untuk analisis data sederhana

Selain itu, penerapan OOP membantu dalam mengorganisasi logika program agar lebih modular, rapi, dan mudah dikembangkan.

## Cara Menjalankan

Jalankan masing-masing file menggunakan Python:

```bash
python tugas4.py
python tugas5.py
python tugas6.py
```

Untuk `tugas6.py`, pastikan dependensi sudah terinstal:

```bash
pip install numpy pandas
```

## Catatan

- Semua tugas yang dibuat merupakan latihan konsep dasar Python
- File `ringkasan_tugas6.txt` akan dihasilkan secara otomatis dari `tugas6.py`
